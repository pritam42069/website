Installation

On windows, go to python.org and download the installer, click, and install.

On linux based systems, refer to your respective package managers. Note that most linux distributions come preinstalled with python, refer to the next paragraph to verify if it is installed.

On mac, install homebrew first and then install python.

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"

brew install python

If you think you fucked up something somewhere, refer to this installation guide.

To verify that python has be successfully installed, open your terminal (win + r and then enter `cmd` for windows) and type python --version.

If the output is something like Python 3.x.x, you are good to go.
